Free blocks for Elementor page builders.

== Description ==

Layoutist offers free block templates for Elementor page builders. Other page builder will be supported soon.

== Installation ==

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Templates.
4. Import any template your like.
